# Futuristic Risk Intelligence

**[futuristicrisks.com](https://futuristicrisks.com)** — 全球宏观风险的量化监测与分析

## What is this

Geopolitical risk intelligence platform that combines verified news intelligence with probabilistic modeling to provide:

- **Real-time risk probabilities** for 6 major conflict regions (updated daily)
- **Cascade impact analysis** — how conflicts transmit through energy, shipping, and supply chains to specific industries and assets
- **Financial impact calculation** — from event probability to your P&L
- **AI agent integration** via MCP/API — turn your AI into a risk analyst

## Data Feed

Risk data available via MCP or JSON API:

```bash
pip install war-dashboard-data
```

